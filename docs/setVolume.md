# $setVolume
Set the volume of the music player.
## Usage
```
$setVolume[amount]
```
## Fields
|  Name  |           Description            |  Type  | Required | Rest |
|--------|----------------------------------|--------|----------|------|
| Amount | The volume amount to be applied. | Number | Yes      | No   |
