# $enableAllFilters
Enable the provided FFMPEG filters.
## Usage
```
$enableAllFilters[...filters?]
```
## Fields
|  Name   |         Description         | Type | Required | Rest |
|---------|-----------------------------|------|----------|------|
| Filters | Filter names to be enabled. | Enum | No       | Yes  |
