# $isFilterEnabled
Check whether the provided filter is enabled.
## Usage
```
$isFilterEnabled[filters]
```
## Fields
|  Name   |         Description         | Type | Required | Rest |
|---------|-----------------------------|------|----------|------|
| Filters | Filter names to be toggled. | Enum | Yes      | No   |

## Output
> Boolean
