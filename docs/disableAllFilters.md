# $disableAllFilters
Disable the provided FFMPEG filters.
## Usage
```
$disableAllFilters[...filters?]
```
## Fields
|  Name   |         Description          | Type | Required | Rest |
|---------|------------------------------|------|----------|------|
| Filters | Filter names to be disabled. | Enum | No       | Yes  |
