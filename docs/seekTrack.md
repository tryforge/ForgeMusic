# $seekTrack
Seeks a track.
## Usage
```
$seekTrack[duration]
```
## Fields
|   Name   |         Description          | Type | Required | Rest |
|----------|------------------------------|------|----------|------|
| Duration | Seek duration to be applied. | Time | Yes      | No   |

## Output
> Boolean
