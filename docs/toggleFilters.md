# $toggleFilters
Toggle the provided FFMPEG filters.
## Usage
```
$toggleFilters[...filters?]
```
## Fields
|  Name   |         Description         | Type | Required | Rest |
|---------|-----------------------------|------|----------|------|
| Filters | Filter names to be toggled. | Enum | No       | Yes  |
