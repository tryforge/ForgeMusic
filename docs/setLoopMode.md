# $setLoopMode
Set the loop mode of the music player.
## Usage
```
$setLoopMode[mode]
```
## Fields
| Name |            Description             |  Type  | Required | Rest |
|------|------------------------------------|--------|----------|------|
| Mode | The loop mode of the music player. | String | Yes      | No   |
