# $skipTo
Skip the current track to the given position.
## Usage
```
$skipTo[position]
```
## Fields
|   Name   |           Description            |  Type  | Required | Rest |
|----------|----------------------------------|--------|----------|------|
| Position | The track position to be played. | Number | Yes      | No   |

## Output
> Boolean
