# $removeTrack
Removes the track that is located at the given position.
## Usage
```
$removeTrack[position]
```
## Fields
|   Name   |            Description            |  Type  | Required | Rest |
|----------|-----------------------------------|--------|----------|------|
| Position | The track position to be removed. | Number | Yes      | No   |

## Output
> Boolean
